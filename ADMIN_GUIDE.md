# 🛠️ TaskQuest 管理者ガイド

**マネージャー・管理者向け操作マニュアル**

このガイドは、MANAGER または ADMIN 権限を持つユーザー向けの管理機能の使い方を説明します。

---

## 👤 権限について

| ロール | できること |
|--------|------------|
| **USER** | タスク実行・報酬交換・プロフィール編集 |
| **MANAGER** | + プロジェクト・エピック・タスクの作成・編集、チーム管理 |
| **ADMIN** | + 報酬管理、バッジ管理、ユーザー管理、全機能アクセス |

---

## 📁 プロジェクト管理

### プロジェクトを作成する

1. 「**プロジェクト**」メニューをクリック
2. 「**＋ 新規プロジェクト**」ボタンをクリック
3. タイトルと説明を入力
4. 「**作成**」ボタンをクリック

### プロジェクトを編集する

1. プロジェクト詳細ページを開く
2. プロジェクト名の横にある **✏️** ボタンをクリック
3. タイトル・説明を編集
4. 「**保存**」ボタンをクリック

---

## 📌 エピック管理

### エピックを作成する

1. プロジェクト詳細ページを開く
2. 「**＋ エピック追加**」ボタンをクリック
3. タイトルと説明を入力
4. 「**作成**」ボタンをクリック

> 💡 エピックはタスクをグループ化するための中分類です

### エピックを編集する

1. プロジェクト詳細ページでエピックを見つける
2. エピック名の横にある **✏️** ボタンをクリック
3. タイトル・説明を編集
4. 「**保存**」ボタンをクリック

---

## ✅ タスク管理

### タスクを作成する

**方法1: タスク一覧から**
1. 「**タスク**」→「**＋ 新規タスク**」をクリック
2. タスク情報を入力
3. プロジェクトとエピックを選択

**方法2: プロジェクト詳細から（推奨）**
1. プロジェクト詳細ページを開く
2. エピック内の「**＋ タスクを追加**」をクリック
3. プロジェクトとエピックが**自動選択**されます

### タスク作成時の入力項目

| 項目 | 説明 |
|------|------|
| **タイトル** | タスク名（必須） |
| **説明** | 詳細な内容 |
| **優先度** | 緊急 / 高 / 中 / 低 |
| **難易度** | ⭐1〜5（難しいほど高ポイント） |
| **基礎ポイント** | 完了時に獲得するポイント |
| **ボーナスXP** | 完了時に獲得するXP |
| **期限** | 任意 |
| **担当者** | 複数選択可能 |

### 🤖 AI推奨値を使う

タスク作成画面で「**🤖 分析中...**」ボタンをクリックすると、AIがタスク内容を分析して最適なポイントとXPを自動計算します。

1. タイトルと説明を入力
2. 優先度と難易度を選択
3. 「**🤖 AI推奨値**」ボタンをクリック
4. 推奨値が自動入力される（変更可能）

### タスクを編集する

1. プロジェクト詳細ページでタスクを見つける
2. タスク名の横にある **✏️** ボタンをクリック
3. 編集モーダルで以下を変更可能：
   - タイトル・説明
   - 優先度・難易度
   - 基礎ポイント・ボーナスXP
4. 「**保存**」ボタンをクリック

---

## 👥 チーム管理

### チームメンバーを確認する

1. 「**プロジェクト**」メニューをクリック
2. 「**チーム**」タブをクリック
3. メンバー一覧が表示されます

### メンバーの役割を変更する（ADMINのみ）

1. チーム一覧でメンバーを見つける
2. 「**編集**」ボタンをクリック
3. ロール（USER / MANAGER / ADMIN）を変更
4. 「**保存**」ボタンをクリック

---

## 🎁 報酬管理（ADMINのみ）

### 報酬を追加する

1. 「**報酬**」→「**報酬管理**」をクリック
2. 「**＋ 報酬追加**」ボタンをクリック
3. 報酬情報を入力：
   - 報酬名
   - 説明
   - カテゴリ
   - 必要ポイント
   - 在庫数（無制限にする場合は空欄）
4. 「**作成**」ボタンをクリック

### 報酬を編集・削除する

1. 報酬管理ページで報酬を見つける
2. 「**編集**」または「**削除**」ボタンをクリック

### 交換申請を処理する

1. 「**報酬**」→「**交換申請**」をクリック
2. 申請一覧から処理するものを選択
3. 「**承認**」または「**却下**」ボタンをクリック

---

## 🏅 バッジ管理（ADMINのみ）

### バッジを作成する

1. 「**バッジ**」メニューをクリック
2. 「**＋ バッジ追加**」ボタンをクリック
3. バッジ情報を入力：
   - バッジ名
   - 説明
   - アイコン
   - 条件タイプ（タスク完了数 / ログイン連続日数 / 累計ポイント / レベル）
   - 条件値
4. 「**作成**」ボタンをクリック

---

## 📦 アーカイブ管理

完了したプロジェクト・エピック・タスクを整理できます。

### アーカイブする

1. 「**アーカイブ**」メニューをクリック
2. 「**＋ 完了済みをアーカイブ**」ボタンをクリック
3. アーカイブ候補が表示される
4. 個別に「**アーカイブ**」または「**📦 一括アーカイブ**」

### アーカイブ解除する

1. アーカイブ一覧から解除したいアイテムを見つける
2. 「**解除**」ボタンをクリック

---

## 💡 管理者向けTips

### 効率的なタスク管理

1. **プロジェクト構造を先に作る** - 先にプロジェクト→エピックの骨組みを作成
2. **AIを活用** - ポイント設定はAI推奨値を参考に
3. **期限を設定** - 期限があるとユーザーの優先順位が明確に
4. **適切な難易度設定** - 難易度に応じたポイントでモチベーション維持

### チームのモチベーション維持

1. **魅力的な報酬を用意** - ポイントを貯める動機に
2. **バッジで称賛** - 努力を可視化
3. **ランキングを活用** - 適度な競争心を刺激
4. **定期的なアーカイブ** - 完了済みを整理して達成感を演出
